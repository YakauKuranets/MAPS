# Плейбук: Обнаружение узла "Синдром"

Если модуль `radio_hunter.py` подсвечивает на 3D-Глобусе красную цель (Аномальная вышка / IMSI-Catcher):

## Немедленные действия оператора (Тайминг: < 30 секунд):
1. **Подтверждение:** Выделить цель на карте (WebGPU Terminal). Проверить граф в Neo4j на наличие связей с известными провайдерами.
2. **Изоляция:** Выделить тактическую зону радиусом 2 км вокруг цели.
3. **Команда Агентам:** Отправить через Mesh-сеть приказ `executeIdentitySwitch()` всем мобильным агентам в зоне.

## Эскалация (Тайминг: < 5 минут):
1. Активировать протокол `Отравленный Колодец` в данной гео-зоне.
2. Ожидать перехвата дампов через Userbot Синдиката (Telegram).
3. Доложить Создателю через зашифрованный канал.
