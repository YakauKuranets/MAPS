# Плейбук: Отравленный колодец

Этот раздел описывает только оборонительное моделирование (симуляции и тестовые сценарии) для повышения устойчивости системы.

1. Запустить генератор синтетических сущностей в тестовом контуре.
2. Отслеживать влияние на детекторы и систему корреляции.
3. Зафиксировать метрики и откатить симуляцию.
