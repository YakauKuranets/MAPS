# eBPF-щит (Ring 0)

Модуль **eBPF Shield** обеспечивает мониторинг и фильтрацию трафика на уровне ядра Linux, используя технологии Cilium и Tetragon.

## Назначение

- Обнаружение сетевых аномалий (сканирование портов, подозрительные соединения) за микросекунды.
- Блокировка атакующих IP прямо на сетевом интерфейсе до того, как трафик достигнет приложения.
- Сбор телеметрии для последующего анализа в Neo4j.

## Развёртывание

eBPF-программы загружаются в ядро автоматически при старте каждого узла Kubernetes через демон Cilium. Правила фильтрации задаются в манифестах `CiliumNetworkPolicy`.

## Интеграция с SOAR

При срабатывании детектора генерируется событие, которое отправляется в Aegis SOAR для автоматического реагирования (например, добавление IP в блок-лист Cloudflare).
