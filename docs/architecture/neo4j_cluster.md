# Кластер Neo4j

Neo4j используется как графовая база для корреляции событий, атрибуции угроз и поиска аномалий по оперативным сущностям.

## Ключевые роли
- Хранение графа `Operative -> INTERCEPTED_SIGNAL -> CellNode`.
- Быстрые запросы для `radio_hunter.py`.
- Поддержка аналитики связей в командном интерфейсе.
